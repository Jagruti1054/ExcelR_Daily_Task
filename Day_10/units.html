CSS units

It is used to define the size of an element

They are of two types

  1. Absolute Units
  2. Relative Units

Absolute Units

This are fix units

  • px-pixel
  • cm-centimeter
  • mm-millimeter
  • in-inch
  • pt-point
  • pc-pica

Relative Units

  • %-percent
  • em-em
  • rem-root em
  • vw-viewport width
  • vh=viewport height
